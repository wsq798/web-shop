# web-shop
这是我的第一个web项目。
## 项目简介
我将完成一个简单的web商城，包括用户注册、登录、商品展示、购物车、订单、支付等功能。

## 技术栈
- **前端**：HTML、CSS、JavaScript
- **后端**：php、mysql
- **服务器**：Apache、Nginx
- **数据库**：MySQL
- **版本管理**：Git
